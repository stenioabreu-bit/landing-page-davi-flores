# Davi Flores - Landing Page

Landing page moderna e delicada para a floricultura Davi Flores, com 15 anos de experiência em coroas de flores naturais e arranjos especiais.

## 🎨 Paleta de Cores

- **Lilás**: #A09BE7
- **Creme**: #FFEECF
- **Vermelho**: #B10F2F
- **Preto**: #1A0C1B
- **Verde**: #013E1F
- **Branco**: #FFFDF9

## 🚀 Funcionalidades

- Design responsivo
- Animações suaves ao scroll
- Integração com WhatsApp para compras
- Sistema de filtros de produtos (Todos, Arranjos, Vasos, Coroas)
- Navegação smooth scroll
- Seção de serviços 24h
- Promoções especiais
